## 2. Methodological Questions

### 2.1 What Exact Dataset Fed the Negative Binomial Model?

**Reviewer Concern:** Need one explicit answer: 10,920? 28,756? 75,894?

**Response:** We now provide explicit dataset specification.

**Answer:** The authoritative model used the **canonical dataset**:
- **Records:** 1,183 state-day aggregates
- **Source:** Claude Code processed (vibe indices, z-scored)
- **States:** 13 (7 battleground, 3 control, 3 watch)
- **Terms:** Pre-aggregated into indices (not individual terms)
- **Excluded:** US national rows

Earlier analyses using raw data (75,894 records) have been **deprecated**.

---

### 2.2 Why Negative Binomial Given Extreme Zero Structure?

**Reviewer Concern:** Why NB not zero-inflated given 50-79% zeros?

**Response:** NB was chosen for parsimony, but zero-inflated is reasonable.

**Defense:**
- NB handles moderate zero inflation and overdispersion
- With 28,756 records (original analysis), NB was conservative
- Canonical dataset (1,183) uses pre-aggregated indices, reducing zero issue
- **Acknowledgment:** Zero-inflated NB would be reasonable sensitivity analysis

**Change Made:**
- ✅ Added note: "NB chosen for parsimony; zero-inflated models would be reasonable alternative"
- ✅ Flagged for sensitivity testing in future work

---

### 2.3 Category-Level Interpretations Without National Contrasts
