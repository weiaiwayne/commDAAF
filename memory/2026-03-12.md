- **Problem**: 64% of search results mention AI but aren't about AI
- **Solution**: Created `filter_transcripts.py` with AI density scoring

### False Positive Context Analysis ✅
AI mentioned in non-AI hearings because:
1. List of technologies (32%): "5G, AI, cloud..."
2. Government modernization (27%): AI as tool
3. National security brief mention (15%)
4. Workforce/automation (10%)
5. Specific application (10%)
6. Budget line items (6%)
