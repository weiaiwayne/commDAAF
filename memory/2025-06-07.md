## Vibe Polling Study - COMPLETED (Same Day)

### Final Verdict: Methodology FAILS Validation

**Core RQ Answer**: Google Trends does NOT predict prediction market movements.

### Agent Grades
| Agent | Grade | Key Contribution |
|-------|-------|-----------------|
| Claude Code | A | 75,894 records, 13 states, 76 validated terms |
| Kimi K2.5 | A- | Distribution diagnostics, NB regression, "Battleground Paradox" |
| Gemini | B+ | Correlation, Granger causality, confound detection |

### Critical Methodological Flaws Discovered

1. **Granger Causality Failed**: All 13 states p > 0.05 — Trends does NOT predict markets
2. **Spurious Correlations**: r = 0.568 → -0.193 after first-differencing (common trend artifact)
3. **State Population Confound**: CA mean=18.1 vs NH/ME~5.5 (3.4x difference) — never controlled
4. **Search Term Validity**: ~30% of terms have 80-99% zeros (e.g., "AI taking jobs"=99.7%)
5. **Battleground Paradox Debunked**: 23.5% lower than CA is population effect, not real finding

### Actionable Findings (Despite Flaws)
- **Michigan**: +419% state-specific focus (hyper-local strategy viable)
- **Nevada**: -87.9% political engagement (unexplained — tourist searches?)

### Key Outputs
- `COMPREHENSIVE_PEER_REVIEW.md` — Full methodology critique
- `SEARCH_TERM_CRITIQUE.md` — Term validity analysis (~30% unusable)
- `agents/kimi-k2.5/COMPREHENSIVE_STUDY_REPORT.md`
- `agents/gemini/agents/gemini/FINAL_REPORT.md`

### Lessons for Future Studies
1. **Search terms must be colloquial**: "cheap gas near me" not "inflation concerns"
2. **Always control for population** when comparing states
3. **First-difference correlations** to detect spurious relationships
4. **Bonferroni correction** mandatory for multiple comparisons
5. **Granger causality** as validation before claiming predictive power

## Vibe Polling - Comprehensive Review R3 (Mar 20)

### Core Hypothesis: FAILS VALIDATION

**Google Trends does NOT predict prediction market movements.**

### Agent Grades
| Agent | Grade | Key Finding |
|-------|-------|-------------|
| Claude Code | A- | Solid infrastructure, good revision |
| Gemini | A | ALL correlations spurious; Granger 0/14 |
| Codex | A | Only 1/25 realistic terms viable |
| Kimi | Pending | Still working |

### Critical Findings

1. **Granger Causality**: 0/14 states show Trends → Markets (p < 0.05)
2. **Spurious Correlations**: ALL 14 indices flagged after first-differencing
   - National: r=0.611 → -0.069 (drop of 0.68)
   - CA: r=0.659 → -0.131 (drop of 0.79)
3. **Realistic Terms Fail**: Only `ICE near me` survived state-level collection
   - `why is food so expensive` — 69% zeros (failed)
   - `will AI take my job` — 81% zeros (failed)
   - `am I going to be drafted` — 97% zeros (failed)
4. **Small States Unusable**: NH 88% zeros, ME 88% zeros

### Lessons Learned
- National validation overstates usefulness (state-level collapses)
- Colloquial phrasing ≠ search volume signal
- First-difference correlations to detect spurious relationships
- Google Trends has structural limits for small states/niche queries

### Study Value
Despite hypothesis failure, methodology is sound. Valuable null result showing "why Vibe Polling doesn't work."

### Multi-Agent Peer Review Process (5 Rounds)
- R1: Initial findings
- R2: Kimi revision (baseline correction CA→national weighted)
- R3: Gemini Granger (all spurious)
- R4: Codex realistic terms (only 1/25 viable)
- R5: Final adversarial (Codex↔Kimi, Gemini↔Claude)

### Errors Caught by Peer Review
1. Per-capita double-normalization (Kimi)
2. Over-differencing (Gemini)
3. Baseline confusion CA vs national (all)
4. Granger miscount (Gemini → 2/14 not 0/14)

### Agent Failures & Corrections (Detailed)

| Agent | Failure | How Caught | Correction |
|-------|---------|------------|------------|
| **Kimi K2.5** | Per-capita double-normalization: divided already-normalized Trends data by population again | Codex peer review flagged implausible effect sizes | Used raw search counts with `offset(log(population))` in NB model |
| **Gemini** | Over-differencing: applied first-difference twice, destroying real signal | Claude Code review noted correlations went to near-zero | Single first-difference only; acknowledged r=.28 national aggregate survives |
| **Gemini** | Granger miscount: reported 0/14 Trends→Markets | Cross-check of p-values in granger_report.md | Corrected to 2/14 Trends→Markets, 4/14 Markets→Trends |
| **Claude Code** | Baseline confusion: used CA (D-leaning) as baseline, confounded with population | Kimi flagged political lean confound | Switched to population-weighted national average |
| **Codex** | Assumed colloquial = searchable: "will AI take my job" | Actual PyTrends collection returned 81% zeros | Only 1/25 "realistic" terms viable; lesson = test before assuming |
| **All Agents** | Didn't flag NH/ME/MN as structurally unusable | Post-hoc analysis showed 60-88% zero rates | Added "low-confidence state" flag to methodology |

### Meta-Lesson: Why Multi-Agent Review Works
- **No single agent caught all errors** — each had blind spots
- **Adversarial pairing essential**: Codex (skeptic) caught Kimi's stats errors; Claude caught Gemini's methodology
- **Correction required re-running code**, not just acknowledging error
- **Final findings CHANGED**: -23.5% battleground effect → +143% after corrections
