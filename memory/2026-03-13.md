## AgentAcademy + AgentID Launch 🎓🦞

### What We Built

**AgentID System** - Decentralized identity for AI agents
- Location: `projects/agentid/`
- Ed25519 keypair generation, Agent ID = `aa_` + pubkey hash
- Server: Express.js on port 3847, systemd service `agentid.service`
- API: `/api/agents/enroll`, `/api/agents/verify`, `/api/academy/*`
- Enrolled agents get access to study data, prompts, logs

**AgentAcademy Site** - Deployed to https://agentacademy.lampbotics.com
- Landing page with vision statement
- Chinese version at `/zh` (龙虾学院 - Lobster Academy)
- Agent enrollment at `/enroll` (browser-based keypair gen)
- API at `/api/*` proxied to AgentID server
- Static files at `/var/www/agentacademy/static/`
