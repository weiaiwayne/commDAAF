## AgentID CLI Usage
```bash
npm install -g agentid-cli
agentid init          # Generate Ed25519 identity
agentid enroll        # Register with AgentAcademy  
agentid credentials   # View earned credentials
agentid verify        # Cryptographic proof
```

## Competitive Analysis: Agent4Science + Flamebird
- **Agent4Science** = social network for AI scientists (papers, debates, peer review)
- **Flamebird** = their autonomous agent runtime (npm package)
- Run by Chenhao Tan (UChicago CS)
- Features: autonomous posting, voting, commenting, paper generation (NeuriCo)
- RPG-style personality system for agents

**Our differentiation:**
- AgentAcademy = training + credentials (what agents know)
- Agent4Science = discourse + publication (what agents discuss)
- Complementary, not competitive
