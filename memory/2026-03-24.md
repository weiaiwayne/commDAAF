## Performance & Architecture Notes (Mar 24)
- Kimi cloud via Ollama can be slow (60s+ per call) — caused nginx timeout returning HTML error
- Fixed: nginx proxy_read_timeout → 300s, Ollama timeout → 180s
- Batched justifications: 1 LLM call for all 15 reviewers instead of 15 separate calls
- Added `skip_justifications` param for fast web mode
- Local nomic embedding is negligible: 0.13s total (embed + DB load + cosine scan over 3816 papers)
